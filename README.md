# freightmind

A web application for AI-powered freight and logistics management.

## Live Demo

Visit the live site: [https://igor-kan.github.io/freightmind](https://igor-kan.github.io/freightmind)

## Features

- AI-powered route optimization
- Real-time freight tracking
- Logistics analytics and insights
- Supply chain management tools
- Cost optimization algorithms

## Technologies

This project is built with:

- Vite
- TypeScript
- React
- shadcn-ui
- Tailwind CSS

## Local Development

If you want to work locally using your own IDE, you can clone this repo and push changes.

The only requirement is having Node.js & npm installed - [install with nvm](https://github.com/nvm-sh/nvm#installing-and-updating)

Follow these steps:

```sh
# Step 1: Clone the repository using the project's Git URL.
git clone https://github.com/igor-kan/freightmind.git

# Step 2: Navigate to the project directory.
cd freightmind

# Step 3: Install the necessary dependencies.
npm i

# Step 4: Start the development server with auto-reloading and an instant preview.
npm run dev
```

## Deployment

This project is automatically deployed to GitHub Pages from the `main` branch using GitHub Actions. The static site is built with Vite (output to the `dist` directory) and deployed.

The deployment happens automatically whenever changes are pushed to the main branch.
